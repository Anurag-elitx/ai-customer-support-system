# Design Taste & High-End Aesthetics
This skill ensures all UI/UX within the "AI Customer Support System" maintains a premium, "wow-factor" visual style.

## High-Quality Design Checklist
1. **Asymmetry**: Offset layout elements to create dynamic movement.
2. **Neon Glow**: Use subtle outer glows for primary actions (Indigo/Purple).
3. **Glassmorphism**: Backdrop-blur is essential for creating depth and premium layering.
4. **Typography Contrast**: Extreme weight and size differences (e.g., Bold `display-md` headers vs. Semi-thick `label-xs` tags).
5. **Breathing Interactions**: Buttons and Orbs should have a subtle, slow pulse when idle.

## Role of the Agent
- Use this skill to evaluate generated screenshots for "visual excellence."
- Ensure all screens feel cohesive, premium, and sophisticated.
- Reject any designs that feel flat or generic.
