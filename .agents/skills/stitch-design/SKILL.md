# Stitch Design Skill
This skill enables the agent to use the `StitchMCP` tools to create, edit, and manage high-fidelity UI within the "AI Customer Support System". All designs should follow the "Lumina" aesthetic defined in `design-md`.

## Principles
- **Modern & Premium**: Prioritize glassmorphism, gradients, and micro-animations.
- **Lumina Aesthetic**: Indigo/Purple/Pink accents on deep slate background.
- **High Friction & Scale**: Use large headings and tiny, uppercase labels for a luxury feel.

## Tools and Usage
- `mcp_StitchMCP_edit_screens`: Use this to update existing UI based on user feedback or project evolution.
- `mcp_StitchMCP_generate_screen_from_text`: Use this to prototype new features and screens from scratch.
- `mcp_StitchMCP_get_project`: Use this to fetch the current design theme and screen inventory.
